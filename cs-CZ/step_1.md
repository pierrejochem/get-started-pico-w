## Úvod

Raspberry Pi Pico W je produkt Raspberry Pi, který přidává k Raspberry Pi Pico WiFi funkce, což umožňuje připojení zařízení k WiFi síti. V této příručce se naučíš, jak používat Raspberry Pi Pico W, jak jej připojit k WiFi síti a jak jej proměnit ve webový server pro ovládání digitálních výstupů z prohlížeče a příjem dat ze senzorů.

![Raspberry Pi Pico W je ovládán z webové stránky.](images/web-server.gif)

<p style="border-left: solid; border-width:10px; border-color: #0faeb0; background-color: aliceblue; padding: 10px;">
<span style="color: #0faeb0">**Wi-Fi**</span> Zařízení s podporou Wi-Fi umožňují snadnou komunikaci mezi počítači a internetem. Tato technologie pomohla nastartovat revoluci internetu věcí (IoT).
</p>

V tomto projektu:

- Připoj Raspberry Pi Pico W k WiFi hubu
- Vytvoř webový server na Raspberry Pi Pico W pro zobrazení webové stránky
- Použij svou webovou stránku k ovládání LED diody Raspberry Pi Pico W a k získání teplotních dat

Co budeš potřebovat:

- Raspberry Pi Pico W a datový kabel mikro USB
- Počítač připojený k vaší síti
- Thonny Python IDE
